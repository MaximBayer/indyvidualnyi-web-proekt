# Навчальний вебпроєкт (PHP + MySQL)

Функціонал:
- список записів з БД;
- детальна сторінка запису;
- фільтрація по категоріях;
- адмін-панель з CRUD;
- авторизація в адмінці.

## Структура
- `index.php` - список новин
- `post.php` - сторінка конкретного запису
- `category.php` - список за категорією
- `function.php` - PHP-функції (DB/CRUD)
- `conf.php` - підключення до MySQL
- `admin/` - адміністративна панель
- `login/` - вхід до адмінки
- `sql/project.sql` - структура БД + демо-дані

## Швидкий старт (локально)
1. Створи БД через `sql/project.sql`.
2. В `conf.php` вкажи свої параметри MySQL.
3. Запусти сайт через Apache (XAMPP/OpenServer).

## Адмінка
- URL: `/login/index.php`
- Логін: `admin`
- Пароль: `123`
